# Graphify Integration
Antigravity, you have the `graphify` MCP server tools available (`query_graph`, `get_node`, `get_neighbors`, etc.). For general codebase queries, relationship extraction, or large scale architecture questions, use the graphify tools instead of indiscriminately reading files. This helps in massive token savings. 

When generating the codebase structure, first consult graphify using its provided MCP tools.
